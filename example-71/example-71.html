CSS Image Tutorial

Example #1: Image Sizes

You can control the size of an image using the width and height properties in CSS, just like you can for any box. You specify image sizes because it helps pages to load smoothly because the HTML and CSS code will often load before the images, the height and width properties tell the browser how much space to leave for an image while the rest of the page loads.

Example #2: Aligning Images

The vertical-align attribute refers to how to align text with the image.

Valid values for the vertical-align attribute are "top", "middle", and "bottom".

The values: "top", "middle", and "bottom" refer to what part of the image to align the first line of text with.

For the image to align left or right use the CSS property float. For the image to align to the center, we need to use the display and margin properties.
